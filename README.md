## How to Run the App

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-username>/<your-repo>.git
   cd <your-repo>
   
2. **Install dependencies**
 npm install

3. **Start the development server**
   ```bash
   npm run dev

4. **Open in Browser**
   ```bash
   http://localhost:3000
